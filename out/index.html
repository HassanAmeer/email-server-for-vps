Robust Architecture

Designed for Programmatic Mail Testing

A self-hosted API node is perfect for automating registration, password recovery, and email verification testing.

Programmatic REST API

Connect and control your mailboxes completely via HTTP APIs. Instantly query incoming mail objects, sender metadata, and raw headers in standard JSON.

Automated OTP Parsing

The parser automatically scans inbound mail subjects and body texts for 4-6 digit numeric authentication codes. Retrieve clean verification codes instantly.

Zero-Log VPS Privacy

Host the server on your own node. Email records, raw bodies, and attachments remain entirely under your local file paths. No external tracking logs.

MIME Attachments

Integrated SMTP parser automatically isolates files from base64 streams and saves them to local storage. Retrieve raw media resources directly over API.

Transient Routing

Generate random mailbox prefixes on demand. The server captures and routes emails to dynamic addresses instantly without prior inbox setup.

DKIM Compliance

Integrated support for generating cryptographically signed DKIM signatures. Easily test bidirectional mailing integrations with full SPF verification.
